## 오늘 연습한 Claude Code 스킬

| 스킬 | 하는 일 |
|------|---------|
| `/context` | 현재 컨텍스트 사용량을 색 그리드로 시각화 — 얼마나 찼는지 확인 |
| `/resume` | 이전 세션 목록에서 골라 그 컨텍스트 그대로 이어가기 |
| `/model` | 사용할 모델 전환하고 기본값으로 저장 |
| `/effort` | 모델 사고 강도(reasoning effort) 설정 — low · medium · high · xhigh · max (또는 auto) |
| Plan 모드 (`Shift+Tab`) | 모드 순환 토글. Plan 모드에선 코드를 바로 고치지 않고 계획을 먼저 세워 보여준 뒤, 승인하면 실행 |
| `/statusline` | 터미널 하단 상태줄 커스터마이즈 — **토큰 사용량 표시**도 여기서 설정 |
| `/init` | 코드베이스를 훑어 `CLAUDE.md` 초기 생성 |

### 헷갈리기 쉬운 것 — `/clear` vs `/context`

- **`/context`** — 컨텍스트가 얼마나 찼는지 **보여주기만** 한다. 아무것도 지우지 않음.
- **`/clear`** — 대화를 비우고 **완전히 새로운 세션**을 시작한다. CLI에서는 이게 새
  세션으로 취급되어, 이전 세션은 그대로 보존되고 `/resume` 목록에 **별도 항목**으로
  나뉜다.
